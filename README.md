# Plopper

This package help you generate your react and redux files in your project

## How to Use

`npm install --save-dev plopper`

Then

`npx plopper`

## `.plopper.json` file
```json
{
    "bypass": {
        "component": {
            "useCssModule": "true"
        }
    }
}
```
